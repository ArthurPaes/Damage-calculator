## Installation

### Clone

- Clone this repo to your local machine using `https://github.com/ArthurPaes/Damage-calculator.git`
```shell
$ git init
$ git clone
```
### Setup


> Execute `yarn` or `npm instal` to install all the dependencies

```shell
$ yarn 
$ brew install fvcproductions
```
